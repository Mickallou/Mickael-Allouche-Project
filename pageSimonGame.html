Simon Game


In this HTML, CSS, and JavaScript project,
I created a Simon game where the computer generates unique sequences in each round.
Players engage in a memory challenge,
replicating the ever-changing patterns presented by the computer.
The project seamlessly integrates front-end technologies to deliver
an interactive and dynamic gaming experience,
keeping each gameplay session distinctive and engaging.
