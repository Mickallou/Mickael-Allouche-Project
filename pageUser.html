User Management


In this HTML, CSS, and JavaScript project,
I developed a user management application that utilizes local storage for data persistence.
Users can easily log in, log out, and delete their profiles with dedicated buttons.
The seamless integration of these front-end technologies provides a secure and
user-friendly experience for managing and storing user data locally.
