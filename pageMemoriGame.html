Memorie Game


In this HTML, CSS, and JavaScript project, I developed
a memory card game featuring globally recognized brands.
Players engage in a challenging and entertaining experience
by matching pairs of well-known logos.
The responsive design, powered by HTML and CSS, ensures a seamless gaming interface,
while JavaScript adds interactivity,
creating an engaging memory challenge with iconic international brands.
